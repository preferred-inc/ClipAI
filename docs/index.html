Features

シンプルだから、速い。

余計なUIは一切なし。必要なときに呼び出し、用が済んだら消える。作業の流れを止めないAIアシスタント。

ワンショートカット

⌘ ⌥ I を押すだけでAIが起動。コピーしたテキストを即座にClaude AIへ送信します。

ストリーミング応答

Claude APIからの回答がリアルタイムに表示。待ち時間のストレスがありません。

カスタムプロンプト

用途に合わせてプロンプトを自由にカスタマイズ。翻訳、要約、コードレビューなど、使い方は自由自在。

メニューバー常駐

Dockには表示されず、メニューバーに静かに待機。必要なときだけ現れる控えめなアプリです。

ネイティブSwiftアプリ

Electronではなく、ネイティブSwiftで構築。メモリ消費は最小限、起動は一瞬。Macに馴染む軽さ。

プライバシーファースト

データ収集なし。バックグラウンドでのクリップボード監視なし。APIキーはローカルに保存。

How it works

3ステップ、それだけ。

複雑な設定は不要。直感的な操作で、すぐに使い始められます。

1
📋

コピーする

気になるテキストやコードを、いつも通り ⌘C でコピー。

2
⌨️

⌘⌥I を押す

ショートカットを押すと、クリップボードの中身がそのままClaude AIに送信されます。

3
✨

回答を受け取る

フローティングウィンドウにAIの回答がストリーミング表示。作業を中断する必要はありません。

Privacy

あなたのデータは、あなたのもの。

ClipAIはプライバシーを最優先に設計されています。収集するデータは一切ありません。

クリップボード監視なし

ショートカットを押したときだけクリップボードにアクセス。バックグラウンドでの監視は一切行いません。

アナリティクスなし

利用状況の追跡、テレメトリ、クラッシュレポートの外部送信は行いません。

ローカル保存

APIキーはMacのローカルにのみ保存。会話履歴やログの記録も行いません。

通信先はAnthropicのみ

外部通信はAnthropic API (api.anthropic.com) のみ。他のサーバーには一切接続しません。

プライバシーポリシー

最終更新日: 2026年3月16日

概要

ClipAIは、クリップボードの内容をAnthropic Claude APIに送信してAIによる分析を行うmacOSユーティリティです。あなたのプライバシーを大切にしており、このポリシーではデータの取り扱いについて説明します。

データ収集について

ClipAIは、個人データの収集、保存、開発者への送信を一切行いません。

ClipAIがアクセスするもの

  • クリップボードの内容: キーボードショートカット（⌘⌥I）で明示的にアプリを起動した場合のみ。バックグラウンドでクリップボードを読み取ることはありません。
  • APIキー: Anthropic APIキーはUserDefaultsを使用してMacのローカルに保存され、Anthropic APIサーバー以外には共有されません。

外部に送信されるもの

  • ClipAIを起動すると、現在のクリップボードテキストがAnthropic API（api.anthropic.com）に送信されます。
  • この通信はAnthropicのプライバシーポリシーおよび利用規約に準拠します。
  • Anthropic APIはデフォルトで入力内容をモデルトレーニングに使用しません。

収集しないもの

  • アナリティクスやテレメトリ
  • 外部へのクラッシュレポート送信
  • ユーザーアカウントや登録情報
  • 閲覧履歴やアプリ使用状況の追跡
  • バックグラウンドでのクリップボード監視

データの保存

すべてのデータはMac上にのみ保存されます。APIキーはローカルのアプリ設定に保存されます。会話履歴やログは保存されません。

第三者サービス

ClipAIが通信するのは Anthropic API（api.anthropic.com）のみです。

ポリシーの変更

このポリシーは随時更新される場合があります。変更はアプリのGitHubリポジトリに掲載します。
